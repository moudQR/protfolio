Featured #2

SKIL 1 Project: Cooperative Educational Game

SKIL 1 - Year 1 project

Context & Background

We built an educational game for high-school students to train teamwork. The concept forced players to cooperate because each player controlled only part of the game mechanics.

My Contribution

I worked on implementing cooperative gameplay logic and integrating the wand-based control concept into the user experience.

Realisations

  • Created a two-part game experience: a platformer and a multiple-choice quiz module.
  • Designed cooperative mechanics where both players must combine their partial controls to win.
  • Implemented custom physical input wands to control the game instead of standard mouse input.

What I Learned

  • Hard skills: game logic design, interaction design, and hardware-assisted input integration.
  • Soft skills: teamwork, playtesting communication, and designing for learning outcomes.
